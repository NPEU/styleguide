A note on Blockrow

Currently, l-blockrow does not apply any styles, so it may seem like a redundant container. However, it's value is in it's delimiting effect; it makes it clear that the contents of the element where this class is applied is intended to take a full-width of the screen, and never be changed due to responsiveness.

Primary Content

This pattern should only really be used once per page, where the primary content of the page is text (like an article) as opposed to being composed of different blocks where it's harder to establish and single primary focus.
